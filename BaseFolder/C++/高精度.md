## 高精度算法：

### 数据的接收方法和存贮方法
```
void init(int a[]){                     //传入一个数组
    string s; 
	cin>>s;                             //读入字符串s 
	a[0]=s.length();                    //用a[0]计算字符串s的位数 
	for(i=1;i<=a[0];i++)
	    a[i]=s[a[0]-i]-'0';             //将数串s转换为数组a，并倒序存储 
	}
```

### 高精度数位数的确定
    位数的确定：接收时往往是用字符串的，所以它的位数就等于字符串的长度。

### 进位,借位处理

#### 加法进位：
```	
    c[i]=a[i]+b[i];
    if (c[i]>=10) { 
        c[i]%=10; 
        ++c[i+1]; 
    }
```                         
#### 减法借位：
```    	
    if (a[i]<b[i]){ 
        --a[i+1]; 
        a[i]+=10; 
    }
    c[i]=a[i]-b[i];
```
#### 乘法进位：	
```
    x=0;
    c[i] = a[i]*b[i];
    x = c[i]/10;
    c[i] %= 10;
    c[i+1] += x


```

### 高精度加
```
void add(int a[], int b[]){
	for(int i = 1; i <= l; i++){
		a[i] += b[i];
		a[i + 1] += a[i] / n;       //进位 
		a[i] %= n;
	}
	if(a[l + 1] > 0){               //考虑从最高位进位到最高位的下一位 
		l++;                        //长度++ 
	}
}
```
